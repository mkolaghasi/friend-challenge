التقنية / Tech

البنية

Flutter + تخزين محلي فقط، بدون خوادم خارجية، مع تصميم واجهة مظلم متجاوب.

الأصول: >20 ملف JSON + أيقونات + خطوط + حزم Lottie.

الأداء

رسوميات خفيفة، استخدام محدود للذاكرة، بدون await كثيف في الإطارات الحرجة، تحسين بسيط لتقليل jank.

// Simplified stat increment aFuture increment(key){ final v = (prefs.getInt(key)??0)+1; prefs.setInt(key,v); }

Architecture

Flutter, on‑device storage only. No remote backend. Responsive dark UI design.

Assets: 20+ JSON files + icons + fonts + Lottie packs.

Performance

Lightweight rendering, minimal memory churn, avoiding heavy async inside critical frames, mild layout caching.
