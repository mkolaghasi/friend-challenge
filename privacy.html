سياسة الخصوصية

آخر تحديث: 03 Sep 2025 – Last Updated
ملخص سريع: لا حسابات – لا إعلانات – لا تحليلات – تخزين محلي فقط (SharedPreferences) يمكن حذفه بمسح بيانات التطبيق.

1. البيانات التي نجمعها

لا نقوم بجمع أو تخزين أو مشاركة أي بيانات شخصية. لا توجد حسابات أو تحليلات أو إعلانات.

2. البيانات المحلية

يتم حفظ الدرجات / الإحصائيات / بعض التفضيلات محلياً فقط. احذف التطبيق أو امسح البيانات لإزالتها.

3. الأذونات

  • INTERNET / ACCESS_NETWORK_STATE / ACCESS_WIFI_STATE: للَّعب المحلي (LAN) وفتح روابط البريد.
  • CHANGE_WIFI_MULTICAST_STATE: لاكتشاف اللاعبين على نفس الشبكة.

4. الإشعارات المحلية

أي إشعار يصدر محلياً على الجهاز دون خادم خارجي.

5. خدمات الطرف الثالث

لا شيء فعّال حالياً. Firebase مُعطَّل. سيتم تحديث هذه الصفحة قبل تفعيل أي خدمة خارجية.

6. خصوصية الأطفال

التطبيق ترفيهي عام ولا يجمع بيانات شخصية.

7. الأمان

لا نرسل بيانات للخارج، المخاطر محدودة. أمان جهازك مسؤوليتك.

8. اختياراتك

  • حذف التطبيق أو مسح البيانات لإزالة التفضيلات.
  • عدم فتح جلسات LAN إذا لم ترغب في الظهور للأجهزة المجاورة.

9. التعديلات

قد نحدّث هذه السياسة وسيتم تغيير التاريخ أعلاه.

Privacy Policy

Last Updated: 03 Sep 2025
Quick Summary: No accounts – no ads – no analytics – on‑device storage only (SharedPreferences) removable via app data clear.

1. Data Collected

No personal data is collected, stored, shared, or sold. No accounts, analytics, or ads.

2. Local Storage

Scores / stats / preferences remain on device. Remove by uninstalling or clearing app data.

3. Permissions

  • INTERNET / ACCESS_NETWORK_STATE / ACCESS_WIFI_STATE: LAN sessions & opening mail links.
  • CHANGE_WIFI_MULTICAST_STATE: Peer discovery on the same network.

4. Local Notifications

All notifications are scheduled locally; no remote push used.

5. Third‑Party Services

None active. Firebase components disabled. Page will update before adding any SDK.

6. Children

General audience app; collects no personal data.

7. Security

No outbound transmission means minimal exposure. Device security remains user responsibility.

8. Your Choices

  • Uninstall / clear app data to remove stored preferences.
  • Avoid LAN mode if you do not want local network discovery.
